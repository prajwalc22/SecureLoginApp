# The SecureLoginApp is a simple desktop application built with Python and Tkinter that provides basic username/password login functionality along with an additional feature to encrypt and decrypt messages. The application uses the Fernet symmetric key encryption algorithm from the cryptography library to ensure secure communication.

# Using the Login Screen:

- Enter your username and password in the provided fields.

- Click the "Login" button to proceed.

- Upon successful login, you will be redirected to the "Encrypt and Decrypt Messages" page.

# Encrypting and Decrypting Messages:

- On the "Encrypt and Decrypt Messages" page, enter the text you want to encrypt in the "Enter text to encrypt" field.

- Click the "Encrypt" button.

- The encrypted message will be displayed in the "Decrypted message" field.
